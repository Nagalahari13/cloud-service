# Cloud Based Media File Storage Service

A cloud-based file storage system that allows users to upload, store, and manage media files using a FastAPI backend and a modern frontend.  
The project is currently implemented and tested on **localhost**, with scope for future cloud deployment.

---

##  Features
- Upload media files (images, documents, etc.)
- Store files securely on the server
- Simple and responsive frontend UI
- FastAPI-based backend with REST APIs
- Clean separation of frontend and backend

---

##  Tech Stack

### Backend
- Python
- FastAPI
- Uvicorn

### Frontend
- HTML, CSS, JavaScript
- React (Vite)
